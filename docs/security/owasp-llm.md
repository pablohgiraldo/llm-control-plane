# OWASP for LLM Applications - Controls in this Platform

- Prompt Injection: validator + injection guard stage.
- Sensitive Information Disclosure: PII detection, redaction, least-privilege logs.
- Supply Chain: pin images, vulnerability scans, trusted base images.
- Authentication & Authorization: JWT/API keys, RBAC middleware.
- Abuse & Rate Limiting: Redis-backed rate limits and quotas.
- Observability: structured logs, metrics for anomalous usage.
