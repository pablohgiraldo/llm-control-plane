# Roadmap (Sprint-focused)

Sprint 1 (Foundation)
- Repo scaffolding, docs, dev infra, empty service skeletons.

Sprint 2
- go.mod, config loading, logging/metrics setup, health endpoints.

Sprint 3
- Auth middleware (JWT/API key), basic RBAC, request context handling.

Sprint 4
- Prompt validators (length, PII, injection guard), policy engine skeletons.

Sprint 5
- Router strategies and provider adapter interfaces; basic fallback logic.

Sprint 6
- Audit events persisted to Postgres; metrics surfaced for key pipeline steps.
